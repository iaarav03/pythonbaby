check_studentid Function

Define a function to map StudentID ranges to buckets (1‚Äì5).
Requirements
  • Handle ranges: 81‚Äì85‚Üí1, 86‚Äì90‚Üí2, 91‚Äì95‚Üí3, 96‚Äì100‚Üí4, others‚Üí5
  • Return the correct bucket as an integer
  • Test with few IDs to confirm boundary cases (85/86/90/91/95/96/100)
def check_studentid(student_id): """ Map StudentID ranges to buckets (1-5). Args: student_id (int): Student ID to categorize Returns: int: Bucket number (1-5) """ if 81 <= student_id <= 85: return 1 elif 86 <= student_id <= 90: return 2 elif 91 <= student_id <= 95: return 3 elif 96 <= student_id <= 100: return 4 else: return 5 # Test boundary cases test_ids = [80, 81, 85, 86, 90, 91, 95, 96, 100, 101] for sid in test_ids: bucket = check_studentid(sid) print(f"Student ID {sid} ‚Üí Bucket {bucket}")
