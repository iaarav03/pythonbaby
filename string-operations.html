String Slicing & Indexing

String slicing is one of Python's most powerful features for text manipulation. Understanding indexing and slicing syntax is essential for effective string processing.

Slicing Syntax

String Slicing Syntax
string[start:end:step] # start: beginning index (inclusive) # end: ending index (exclusive) # step: increment between characters

String Slicing Examples

Python - String Slicing
text = "Hello World" print(f'Original string: "{text}"') # Basic slicing print(f'text[0:5]: "{text[0:5]}"') # "Hello" (characters 0-4) print(f'text[6:11]: "{text[6:11]}"') # "World" (characters 6-10) print(f'text[6:]: "{text[6:]}"') # "World" (from index 6 to end) print(f'text[:5]: "{text[:5]}"') # "Hello" (from start to index 4) # Negative indexing print(f'text[-5:]: "{text[-5:]}"') # "World" (last 5 characters) print(f'text[:-6]: "{text[:-6]}"') # "Hello" (all except last 6) print(f'text[-5:-1]: "{text[-5:-1]}"') # "Worl" (from -5 to -2) # Step parameter print(f'text[::2]: "{text[::2]}"') # "HloWrd" (every 2nd character) print(f'text[1::2]: "{text[1::2]}"') # "el ol" (every 2nd char from index 1) print(f'text[::-1]: "{text[::-1]}"') # "dlroW olleH" (reverse string)

String Methods: find() vs index()

Understanding when to use find() versus index() is crucial for robust string processing and error handling.

Method Comparison

Feature find() index()
Return when found Index position Index position
Return when not found -1 ValueError exception
Error handling Check for -1 Try-except block
Use case Conditional logic When substring must exist

find() vs index() Examples

Python - String Search Methods
text = "Hello world! This is a wonderful world of programming!" print(f'Sample text: "{text}"') # find() method - returns -1 if not found pos1 = text.find('world') pos2 = text.find('xyz') # not found print(f"text.find('world'): {pos1}") # Returns 6 print(f"text.find('xyz'): {pos2}") # Returns -1 (not found) # index() method - raises ValueError if not found pos3 = text.index('world') print(f"text.index('world'): {pos3}") # Returns 6 try: pos4 = text.index('xyz') # This will raise an exception except ValueError as e: print(f"text.index('xyz'): ValueError - {e}") # Finding multiple occurrences first_world = text.find('world') second_world = text.find('world', first_world + 1) print(f"First 'world' at: {first_world}") print(f"Second 'world' at: {second_world}") # Practical usage if text.find('python') != -1: print("Found 'python' in text") else: print("'python' not found in text")

Practical String Processing

Python - Advanced String Processing
# Email parsing email = "user@domain.com" at_pos = email.find('@') if at_pos > 0: username = email[:at_pos] domain = email[at_pos + 1:] print(f"Email: {email}") print(f"Username: {username}") print(f"Domain: {domain}") # URL parsing url = "https://www.example.com/path/to/page" protocol_end = url.find('://') if protocol_end != -1: protocol = url[:protocol_end] rest = url[protocol_end + 3:] domain_end = rest.find('/') if domain_end != -1: domain = rest[:domain_end] path = rest[domain_end:] else: domain = rest path = "/" print(f"\nURL: {url}") print(f"Protocol: {protocol}") print(f"Domain: {domain}") print(f"Path: {path}") # Finding all occurrences def find_all_occurrences(text, substring): positions = [] start = 0 while True: pos = text.find(substring, start) if pos == -1: break positions.append(pos) start = pos + 1 return positions text = "the quick brown fox jumps over the lazy dog" all_the = find_all_occurrences(text, 'the') print(f"\nAll occurrences of 'the': {all_the}") # Case insensitive search print(f"Case sensitive: {text.find('THE')}") # -1 print(f"Case insensitive: {text.lower().find('the')}") # 0

String Formatting & Manipulation

String Formatting Examples

Python - String Formatting
# String methods quick reference text = " Hello World " print(f"Original: '{text}'") print(f"strip(): '{text.strip()}'") print(f"upper(): '{text.upper()}'") print(f"lower(): '{text.lower()}'") print(f"title(): '{text.title()}'") print(f"replace('World', 'Python'): '{text.replace('World', 'Python')}'") # String splitting and joining sentence = "Python is awesome and powerful" words = sentence.split() print(f"\nOriginal: {sentence}") print(f"Split into words: {words}") print(f"Join with '-': {'-'.join(words)}") # String validation methods test_strings = ["hello123", "HELLO", "hello", "123", "Hello World"] for s in test_strings: print(f"'{s}': isalnum={s.isalnum()}, isalpha={s.isalpha()}, " f"isdigit={s.isdigit()}, isupper={s.isupper()}, islower={s.islower()}") # Advanced formatting name = "Alice" age = 30 score = 95.5 print(f"\nF-string: Hello, {name}! You are {age} years old with a score of {score:.1f}") print("Format method: Hello, {}! You are {} years old".format(name, age)) print("Old style: Hello, %s! You are %d years old" % (name, age))

String Processing Best Practices

  • Use f-strings for modern, readable string formatting
  • Choose find() for conditional logic, index() when substring must exist
  • Remember that strings are immutable - methods return new strings
  • Use appropriate string methods instead of manual loops for better performance
  • Consider case sensitivity in your string comparisons
  • Use strip() to remove unwanted whitespace from user input
