List Operations and Pitfalls

Modifying a list while iterating can skip elements due to index shifting. Use the safe patterns below.

Why modifying during iteration is dangerous

Python - Iteration Pitfall
arr = [1, 2, 3, 4, 5] print(f"Original array: {arr}") arr_copy = arr.copy() for i in arr_copy: arr_copy.remove(i) print(f"Removed {i}, array now: {arr_copy}") print(f"Final result: {arr_copy}") # [2, 4] - elements were skipped!

Safe patterns

Python - Safe List Modification
# Method 1: Iterate over a copy arr1 = [1, 2, 3, 4, 5] for x in arr1[:]: if x % 2 == 1: arr1.remove(x) print(f"After removing odds (copy): {arr1}") # Method 2: Iterate backwards by index arr2 = [1, 2, 3, 4, 5] for i in range(len(arr2) - 1, -1, -1): if arr2[i] % 2 == 1: arr2.pop(i) print(f"After removing odds (backwards): {arr2}") # Method 3: List comprehension (creates new list) arr3 = [1, 2, 3, 4, 5] arr3 = [x for x in arr3 if x % 2 == 0] print(f"After keeping evens (comprehension): {arr3}")

List Copying Methods

Different ways to copy a list and how identity vs equality behaves.

Python - Copying Lists
original = [1, 2, 3, 4, 5] copy1 = original[:] copy2 = original.copy() copy3 = list(original) print(f"Slice copy: {copy1}") print(f"Built-in copy: {copy2}") print(f"Constructor copy: {copy3}") print(f"original is copy1: {original is copy1}") print(f"original == copy1: {original == copy1}") print("\nNote: arr[:] makes a COPY of arr, not a reference!")

Sorting: sort() vs sorted()

Comparison

MutabilityModifies original listReturns new list

Return ValueNoneNew list
StabilityStableStable
Python - Sorting Examples
nums = [4, 1, 7, 3] print(f"Original: {nums}") nums_copy = nums.copy(); nums_copy.sort(); print(f"After sort(): {nums_copy}") new_sorted = sorted(nums) print(f"sorted() result: {new_sorted}") print(f"Original unchanged: {nums}") words = ["apple", "pear", "kiwi", "banana"] print(f"\nWords: {words}") print(f"Descending: {sorted(words, reverse=True)}") print(f"By length: {sorted(words, key=len)}") print(f"By last letter: {sorted(words, key=lambda w: w[-1])}")

Tuples and Their Properties

Python - Tuples
t1 = (1, 2, 3, 4, 5) print(f"Tuple: {t1}") try: t1[0] = 10 except TypeError as e: print(f"Cannot modify tuple: {e}") t2 = t1[:] print(f"t1 is t2 (same object): {t1 is t2}") a, b, c, d, e = t1 print(f"Unpacked: a={a}, b={b}, c={c}, d={d}, e={e}") tuple_dict = {(1, 2): "coordinates", (3, 4): "another point"} print(f"Tuple as keys: {tuple_dict}")

Sets and Their Operations

Python - Sets
set1 = {1, 2, 3, 4, 5} set2 = {4, 5, 6, 7, 8} print(f"Union: {set1 | set2}") print(f"Intersection: {set1 & set2}") print(f"Difference: {set1 - set2}") print(f"Symmetric difference: {set1 ^ set2}") list_with_duplicates = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4] unique_set = set(list_with_duplicates) print(f"Unique elements from list: {unique_set}")

Dictionaries and Their Methods

Python - Dictionaries
person = {"name": "Alice", "age": 30, "city": "New York"} print(f"Name: {person.get('name', 'Unknown')}") print(f"Country: {person.get('country', 'Not specified')}") print(f"Keys: {list(person.keys())}") print(f"Values: {list(person.values())}") print(f"Items: {list(person.items())}") squares_dict = {x: x**2 for x in range(1, 6)} print(f"Squares dict: {squares_dict}") additional_info = {"country": "USA", "profession": "Engineer"} merged = {**person, **additional_info} print(f"Merged dict: {merged}")

Useful List Methods

Python - List Methods
demo_list = [1, 2, 3, 2, 4, 2, 5] print(f"Count of 2: {demo_list.count(2)}") print(f"Index of first 2: {demo_list.index(2)}") print(f"Index of 2 after position 2: {demo_list.index(2, 3)}") demo_list.append(6) demo_list.insert(0, 0) demo_list.extend([7, 8, 9]) removed = demo_list.pop() demo_list.remove(2) demo_list.reverse() print(f"Final list: {demo_list}; Last popped: {removed}")
